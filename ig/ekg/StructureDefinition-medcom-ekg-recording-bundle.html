Resource Profile: MedComEkgRecordingBundle

Official URL: http://medcomfhir.dk/ig/ekg/StructureDefinition/medcom-ekg-recording-bundle Version: 1.0.0
Draft as of 2026-01-14 Realm: Computable Name: MedComEkgRecordingBundle

A profile for the MedCom ConditionList Bundle resource.

MedCom EKG Recording Bundle

This page describes how the MedCom Ekg Recording Bundle profile is used within the MedCom EKG Recording standard. It inherits from the MedComDocumentBundle. Please note that the version of the Document Implementation Guide referenced by this link may change over time. For the correct inherited version, refer to the dependency list for this Implementation Guide.

The first entry in the Bundle MUST be a MedComEkgRecordingComposition resource.

The Bundle.timestamp MUST be the time of the assemblement of the Bundle.
